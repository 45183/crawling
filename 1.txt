정적크롤링
- 로그인과 같은 사전 작업 없이, 한 페이지 내부에서 원하는 데이터를 수집할 때 사용
- 주소를 통해 데이터를 요청하고, 결과를 받는 것으로 크롤링이 종료
- 한 페이지 내에서 모든 작업이 이루어지기 때문에 속도가 매우 빠르다
- 페이지의 변화가 조금이라도 필요한 경우에는 적용이 힘들기 때문에 수집 대상에 한계가 있다는 단점


동적크롤링
- 정적 크롤링과는 다르게, 입력, 클릭, 로그인 등과 같이 페이지 내부에서 조작이 필요한 데이터를 수집할 때 사용
- 추가적인 조작이 필요하기 때문에 정적 크롤링에 비해 수집 속도가 느리지만, 수집 대상에 한계가 거의 없다는 큰 장점



정적크롤링 - 정적인 데이터 수집
장점 : 수집 속도 빠름
단점 : 수집 대상에 한계 있음

동적크롤링 - 동적인 데이터 수집
장점 : 수집 대상에 한계 없음
단점 : 수집 속도 느림
